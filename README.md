# Lion_School

## Sobre
Projeto criado com o proposito de rever conceitos e tecnicas desenvolvidas na disciplina de Front-End lecionada pelo professor [Fernando Leonid](https://github.com/fernandoleonid).

## Descrição
A aplicação é uma página desenvolvida para Lion School que ao selecionar um curso é exibido os alunos matriculdados nele, seja cursando ou finalizado. Além de quando selecionado um aluno temos as notas que ele obteve em cada disciplina.

## Ferramentas

|-----------|----------------------------|--------|--------------|

## Instalação do projeto
```
git clone https://github.com/Bre01cc/Lion_School.git
```

## Comandos do teste
- Instalação:
```
npm install --dev-save jest
```
- Iniciar teste:
```
npm test
```
